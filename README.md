# IncRepAndManageSystem API Platform

## Overview

The IncRepAndManageSystem API Platform is designed to facilitate efficient management and reporting of incidents within various systems. This API platform provides a structured way for users to input, track, and manage incidents, ensuring that all necessary data is captured and accessible.

## Features
- **Incident Reporting:** Users can submit new incidents with detailed information.
- **Incident Management:** Track the status and updates of reported incidents.
- **User Authentication:** Secure user access to manage incidents.
- **RESTful API:** Easy integration with various systems and platforms.

## Tech Stack
- **Backend:** Node.js with Express framework
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **API Documentation:** Swagger

## Quick Start Instructions
1. **Clone the repository:**  
   `git clone https://github.com/cfvasquez19/IncRepAndManageSystem-api-platform.git`
2. **Navigate to the project directory:**  
   `cd IncRepAndManageSystem-api-platform`
3. **Install dependencies:**  
   `npm install`
4. **Run the application:**  
   `npm start`
5. **Access the API:**  
   Open your browser and go to `http://localhost:3000` to view the API documentation and endpoints.
